# 💎 Gemini Gems Workshop (Okayama 0210)

Google Gemini のカスタムAIエージェント機能「Gems」の作成から、それを実用的なレベルまで育てていくプロセスを体験するためのワークショップ資料です。

## 📋 コンテンツ

### 0. Gemsとは
Gems（ジェムズ）は、Google Gemini を自分好みにカスタマイズできる機能です。
「特定の役割（例：プログラミング講師、旅行プランナー）」や「特定のルール」をあらかじめ指示（プロンプト）として設定しておくことで、毎回長い説明をしなくても、あなたの意図に最適な回答をくれる専用のAIエージェントを作ることができます。

### 3. Gemsを作ってみる
自分のアイデア、日々の困りごとを解決する Gems を形にします。
- **まずは動けばいい**: 完璧な指示を求めず、まずはAIを動かし始めましょう。
- **最初からいいものを作ろうとしない**: 実際に使いながら修正していく前提で、シンプルに始めます。

### 4. Gemsを育ててみる
作成した Gems にフィードバックを与え、精度を高めていきます。
- **満足いくまで育てる**: 自分で何度も試し、出力が期待通りになるまでプロンプトを調整します。
- **TEFCASの輪を回す**: 試行錯誤のサイクルを高速に回して、AIの挙動を最適化します。

#### 🔄 TEFCASサイクル
Gems（AIプロンプト）の改善に最適な学習フレームワークです。

| ステップ | 項目 | 内容 |
| :--- | :--- | :--- |
| **T** | **Trial All** | **全て試す**。一つの指示に固執せず、様々な言い回しや条件を試す。 |
| **E** | **Event** | **出来事**。その指示の結果、AIがどのような回答をしたか。 |
| **F** | **Feedback** | **フィードバック**。回答に対する違和感や、改善すべき点の気づき。 |
| **C** | **Check** | **確認**。何が意図通りで、何がズレたのかを分析する。 |
| **A** | **Adjust** | **調整**。Checkに基づき、プロンプトの指示を修正・追加する。 |
| **S** | **Success** | **成功**。期待通りの回答が得られた、または新しい使い方の発見へ。 |
